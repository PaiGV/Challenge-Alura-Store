📊 Análisis de Optimización de Negocios: Alura Store Latam

📝 Descripción del Proyecto
Este proyecto consiste en un análisis de datos para la toma de decisiones estratégicas de la empresa Alura Store. El objetivo principal es evaluar el desempeño de cuatro tiendas diferentes para identificar cuál de ellas debe ser vendida, permitiendo al propietario (Sr. Juan) reinvertir el capital en nuevos proyectos.

🛠️ Tecnologías Utilizadas
* Python (Google Colab)
* Pandas (Procesamiento y limpieza de datos)
* Matplotlib (Visualización de resultados)

📈 Factores Analizados
Para llegar a una conclusión sólida, se evaluaron los siguientes indicadores:

* Ingresos Totales: Suma de ventas por tienda.
* Popularidad por Categoría: Productos con mayor demanda.
* Satisfacción del Cliente: Promedio de calificaciones.
* Logística: Costo promedio de envío por tienda.
* Rendimiento de Productos: Identificación de artículos más y menos vendidos.

🏆 Conclusión Estratégica
Tras el análisis, se recomendó la venta de la Tienda 4. Aunque otras tiendas presentan retos operativos, la Tienda 4 muestra el menor volumen de ingresos, lo que representa el mayor costo de oportunidad para la empresa.
